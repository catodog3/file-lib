# Библиотека для работы с файлами

## Описание

Библиотека для работы с файлами. Реализованы методы для чтения и записи текстовых файлов, копирования файлов, удаления файлов, поиска файлов по расширению в директории.

## Установка

Команда для установки:
```shell
mvn clean install
```

Пример добавления зависимости:

```xml
<dependency>
    <groupId>com.github.kazaryan</groupId>
    <artifactId>file-lib</artifactId>
    <version>1.0.0</version>
</dependency>
```
